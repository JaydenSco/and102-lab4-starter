Android Lab 4 - Nationalpark Campgrounds 

Submitted by: Jayden Scott

The core requirements for this project will allow a user to see a list of campground descriptions, with the campground name, description, and media photo. When they tap on a specific campground, it will navigate them to a details screen with the same information, along with the campground's latitude/longitude.
Time spent: 1 hour

Required Features

Campgrounds are displayed using the RecyclerView
Can navigate to the Campground Details screen
Campground images are downloaded and displayed using Glide
